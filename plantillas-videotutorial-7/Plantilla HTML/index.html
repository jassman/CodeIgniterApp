Contenido de Ejemplo

Esta plantilla la vamos a estar utilizando en el Curso de CodeIgniter para demostrar la forma que nos presenta este framework para trabajar con llamados diversos a diferentes vistas para conformar una plantilla o adaptar una desarrollada por terceros.

Este modo de trabajo con plantillas que nos brinda CodeIgniter nos permite ahorras bastante tiempo a la hora de llevar a cabo un desarrollo ya que nos permite reutilizar el código repetitivo utilizado en las secciones del sitio web.
